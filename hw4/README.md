# iCampus 大学校园辅助系统

本项目主题是设计一款面向大学校园生活的辅助系统。

## 文件说明

- `design_report.pdf`：设计说明文档，包含角色设定、内容分析、功能设计、关键任务流程、卡片用例模型、原型界面说明和设计评估。
- `prototype.html`：网页原型，可直接用浏览器打开，展示 iCampus 大学校园辅助系统的关键界面。

## 查看方式

1. 打开 `design_report.md` 查看完整设计报告。
2. 打开 `prototype.html` 查看网页设计方案原型。

## 系统定位

系统名称：iCampus 大学校园辅助系统

系统面向学生、任课教师和校园服务人员，覆盖课程教学、校园通知、场地预约、报修反馈、活动报名和消息提醒等场景。设计目标是提供统一入口，减少校园信息分散和事务处理不透明的问题。
